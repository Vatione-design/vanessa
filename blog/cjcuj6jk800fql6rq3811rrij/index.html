安装 php（要安装56，否则 composer 安装不上） & composer 


brew update
brew tap josegonzalez/homebrew-php // 扩展源
brew tap homebrew/versions
brew install php56
brew install josegonzalez/php/composer

安装 Laravel


composer global require “laravel/installer=~1.1”

添加环境变量


vim ~/.bash_profile
添加 PATH=$PATH:~/.composer/vendor/bin
source .bashprofile
